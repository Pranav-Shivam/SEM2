📚 ε-Greedy Theory
▼

🎰 Multi-Armed Bandit

A row of slot machines, each with a different average reward. You don't know which is best. Goal: maximize total reward over time by making smart choices.

🔍 Exploration vs Exploitation

Explore = try new arms to gather information. Exploit = use what you know to pick the best arm. You can't do both fully at once — this is the core tradeoff.

ε-Greedy Rule

Generate r ~ Uniform(0,1). If r < ε → explore (random arm). Else → exploit (highest Q). ε = 0.1 means 10% exploration, 90% exploitation.

Q(a) — Action Value

Your running estimate of arm a's average reward. Updated every time arm a is chosen: Q_new = Q_old + α(R − Q_old).

Sample Average vs Constant α

Sample average (α=1/N) weights all observations equally — good for stationary problems. Constant α weights recent rewards more — better for non-stationary problems.

❌ Why Pure Greedy Fails

ε=0 means only exploitation. If the first arm tried by accident gives a decent reward, the agent picks it forever — never discovering better arms. Exploration breaks this trap.

📚 UCB Theory
▼

Why UCB over ε-Greedy?

ε-greedy explores blindly — random arm regardless of uncertainty. UCB explores intelligently — arms with high uncertainty get a bonus, making them more attractive to try.

The Bonus Shrinks

Every time arm a is chosen, N(a)↑ → bonus shrinks. You become more confident, so it needs less encouragement. The bonus falls fast (linear denominator).

The Bonus Grows with t

Each step, t↑ → ln(t) grows slowly → all bonuses increase slightly. Neglected arms become "stale" and regain attention. But ln is slow, so this effect tapers.

c — Confidence Level

c=0 → pure greedy. c=2 → classic UCB1. Higher c → tries uncertain arms more aggressively. Lower c → sticks to known good arms more.

N(a)=0 Convention

Unvisited arms have N(a)=0 → division by zero → bonus = ∞. Solution: try each arm once first (round-robin) before computing UCB scores.

O(log T) Regret

UCB1 (Auer et al. 2002) achieves O(log T) cumulative regret — matching the Lai & Robbins lower bound. UCB is essentially optimal for stochastic bandits.

📚 Optimistic Initial Values Theory
▼

💡 The Core Idea

Set Q₁(a) = +5 for all arms (wildly optimistic). Use pure greedy (ε=0). Reality always disappoints → pulled arm's Q drops below untried arms → greedy is forced to try the next one. Exploration for free.

😞 Disappointment Mechanism

Pull arm a → reward R ≈ 0 (true mean) → Q drops: 5 + α(0−5) = 3. Now Q(a)=3 < Q(untried)=5 → greedy switches. Every arm gets tried before any settles.

⚖️ Constant α vs Sample Average

With constant α, the initial bias never fully disappears — it decays geometrically as (1−α)ⁿQ₁. With sample average (1/N), the bias vanishes once every arm has been visited once (Robbins-Monro conditions).

✅ When It Works

Stationary problems — true reward means don't change. The optimism is a one-time opening burst: all arms explored early, then exploitation takes over. Zero extra parameters.

❌ Why It Fails (Non-Stationary)

Once inflated estimates decay, there's no mechanism to keep exploring. If true means shift later, the agent won't notice. UCB re-checks uncertain arms forever; optimism only pushes once.

vs UCB

Both avoid random coin flips. UCB tracks per-arm uncertainty on every step — permanent, adaptive re-checking. Optimism is a single initial push — cheap, zero parameters, but spent once estimates settle.
