How the whole course fits together

Ch.1 What is RL? Agent–env loop. Policy / Value / Model. TD seed (Tic-Tac-Toe). ↓ Ch.2 Bandits — strip away STATE. Pure explore/exploit. Q(a), ε-greedy, UCB, optimistic init, gradient bandit / softmax. ↓ Ch.3 Add STATE + transitions → MDP. Bellman Expectation (predict) → Bellman Optimality (control) → Dynamic Programming — needs a MODEL. ↓ Ch.4 Model unknown → Monte Carlo. Learn V/Q from sampled EPISODES. First-visit MC. Q(s,a) needed for control. ε-soft fixes the greedy local-optimum trap. (Off-policy/IS = beyond mid-sem.) ↓ [Next: TD learning — bootstraps like DP, samples like MC]
The one-line test that recurs everywhere → Does my action change the state I'm in next?
No state at all → MAB. State exists but action doesn't change it → Contextual Bandit. Action changes state → Full MDP.

Multi-Armed Bandits

One state only. No transitions. Pure "which of K actions is best?" — the explore/exploit dilemma in its purest form.

Markov Decision Processes & Dynamic Programming

Add state and transitions. Bellman equations. Solving the MDP exactly — but only if you have the model.

04

Monte Carlo Methods

No model? Learn from sampled episodes instead. First-visit prediction, Q-estimation, and why greedy control needs exploration.
