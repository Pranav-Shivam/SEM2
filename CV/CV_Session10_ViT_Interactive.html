The Vision Transformer, one real forward pass at a time

Inspired by bbycroft.net/llm's idea of showing genuine numbers flowing through a real network rather than a cartoon. Draw a tiny 6×6 image below, then step through every stage of a Vision Transformer — patchify → flatten → linear projection → positional embedding → layer norm → multi-head self-attention → residual → MLP → classification head — with the actual matrix math computed live in your browser at every click.

Honesty check: the weight matrices here are small and randomly initialized once (seeded, so they're stable across reloads) — this is not a trained model, and the final "class probabilities" are not meaningful predictions. What is real: every number you see is computed by actual matrix multiplication, softmax, and layer normalization exactly as the architecture defines them — patchify → embed → attend → feed-forward. The mechanism is genuine; the classification is not.

Step 0 · Build a toy 6×6 image

Click a cell to cycle black → gray → white. This 6×6 grayscale image will be split into a 3×3 grid of 2×2 patches (9 patches total), plus one CLS token — 10 tokens altogether, exactly like a real ViT's patch tokens plus its classification token.
Toy class labels used purely for illustration: Cross, Ring, Diagonal. Since the head is untrained, don't expect the softmax at the end to actually pick the "right" one — watch the mechanism, not the answer.
